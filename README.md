# 简历自动填写助手（本地版）


这是一个本地浏览器扩展，用来维护多份简历资料，并在招聘网站或通用在线表单中执行智能填写。

## 当前能力

- 本地维护多份简历，支持 JSON 导入导出
- 设置页支持更完整的字段模型：
  - 个人信息、身份信息、求职意向、技能专长
  - 多条教育背景、工作经历、项目经历
  - 证书、实习、外语、家庭成员、干部任职、获奖、专利、论文
- 弹窗和页面侧边栏统一为一个动作：`智能填写`
- 智能填写会自动：
  - 识别当前页面可填写字段
  - 匹配本地简历资料
  - 执行填写
  - 回显识别结果和匹配预览
- 支持文本框、文本域、下拉框和部分单选/复选场景

## 安装

1. 打开 Chrome，进入 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择目录 `E:\project\resume-autofill-helper`（这里换成你自己的目录）

## 使用

1. 打开扩展设置页，维护你的简历资料
2. 点击“保存当前简历”
3. 打开招聘网站或在线表单页面
4. 点击扩展弹窗里的“智能填写当前页面”，或使用页面右侧边栏里的“智能填写”
5. 提交前再人工检查一遍

## 导入导出说明

导出文件会保存完整的结构化简历数据。导入时支持两种格式：

- 完整的 `profileStore` 结构
- 单份简历对象，扩展会自动包装成一份可用简历

## 自定义字段

设置页里仍然保留“自定义字段”，格式是一行一个：

```text
身份证号=310xxxxxxxxxxxxx
求职状态=随时到岗
英语水平=CET-6
```

当页面字段名称和这些标签接近时，扩展也会尝试自动匹配并填写。
